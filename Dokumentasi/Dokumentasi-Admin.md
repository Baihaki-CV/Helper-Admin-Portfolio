# Dokumentasi Helper
## Tujuan Dokumentasi
Dokumentasi ini dibuat untuk menjelaskan contoh tugas dan tanggung jawab
yang biasa di lakukan pada posisi Helper .

## Tugas Admin
- Menginput dan mengelola data menggunakan Microsoft Excel
- Membuat dan mengedit dokumen menggunakan Microsoft Word
- Mengarsipkan file secara rapi dan terstruktur
- Mengetik data dengan cepat dan akurat

## Skills
- Pengoperasian komputer
- Microsoft Word
- Microsoft Excel

## Hasil Pekerjaan
- Data tersimpan rapi dan mudah dicari
- Dokumen sesuai standar
- Pekerjaan selesai tepat waktu
